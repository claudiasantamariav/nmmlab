About this project

What This Is

This site is the interactive companion to the paper "Rosetta Stone of Neural Mass Models." It was built during a summer internship at Neuroelectrics × BCOM, so that people could explore all six models hands-on — sliders, live plots, parameter sweeps — instead of just reading equations on a page.

The Paper

Castaldo, F., de Palma Aristides, R., Clusella, P., Garcia-Ojalvo, J., & Ruffini, G. (2025). Rosetta Stone of Neural Mass Models. arXiv:2512.10982

It shows that several widely-used neural mass models aren't separate inventions, but connected rungs on a single mathematical ladder.

Built By

Claudia Santamaria and Lucia Ruffini, as co-developers of this site. Claudia built the Phase-only/Kuramoto, HO, and Wilson–Cowan pages; Lucia built the Stuart–Landau, NMM1, and NMM2 pages.

Supervised By

Francesca Castaldo and Giulio Ruffini, as part of the NE × BCOM summer internship program.

Built With

Plain HTML, CSS, and vanilla JavaScript, with Plotly for the interactive charts — no build step, no framework. Hosted on GitHub Pages.
